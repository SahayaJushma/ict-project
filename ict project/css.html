CSS- Cascading Style Sheet

For improving the presentation of HTML document. It used cascading Style Sheet.

CSS use three types:

  1. Various type of Selectors
  2. Add HTML document
  3. Bootstrap

Syntax:

Selector Strings:

Available are,

  1. Type selector
  2. Universal selector
  3. Id selector
  4. Class selector
  5. Group selector

Type Selector:

What the presentation of rule.

P {

text-align : center;

color: red;

}

Universal Selector:

Presentation of hole page. It denoted by *.

* {

color: red;

}

Id Selector:

Represention of id is #.

# {

color: red;

}

Class Selector:

Represention of class is .

  • {

color: red;

}

Group Selector:

h1,h2,p{ text-align:center;

color: red;

}
